- Initial release of ShopAssist RAG
- Multi-source RAG system (products, reviews, policies)
- ChromaDB vector store integration
- OpenAI embeddings and LLM integration
- FastAPI REST API backend
- Streamlit web interface

- Evaluation framework with test queries
